# Music_Store_DataAnalysis
Data Analysis with PostgreSql

Objective:
To demonstrate advanced SQL capabilities by extracting actionable business insights from a relational music store
database. The analysis covers customer behaviour, sales performance, genre popularity, and strategic
recommendations for marketing and inventory management.


Tools & Technologies: PostgreSQL, pgAdmin4, SQL (CTEs, Window Functions, Aggregate Functions, Joins, Subqueries)


Database Schema: A standard music store schema including tables such as customer, invoice, invoice_line, track,
genre, artist, album and employee.
